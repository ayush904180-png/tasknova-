## 1. Future Brand Roadmap: Scaling 5-10 Years
To transition **TaskNova AI** into our permanent corporate brand **Nuance AI**, our brand architecture must scale gracefully across a broad multi-product matrix:

```
                          ┌────────────────────────┐
                          │   Nuance AI Holding    │
                          └───────────┬────────────┘
                                      │
         ┌────────────────────────────┼────────────────────────────┐
         ▼                            ▼                            ▼
┌─────────────────┐          ┌─────────────────┐          ┌─────────────────┐
│  Nuance Play    │          │ Nuance Console  │          │ Nuance Gateway  │
│ Contributor App │          │ Enterprise SaaS │          │ SDK / API Hub   │
└─────────────────┘          └─────────────────┘          └─────────────────┘
```

### Product Family Naming Protocol
1.  **Nuance Play (Mobile & Web Sandbox):** The high-speed contributor micro-task arena. Our gamified but clean ledger-earning node system.
2.  **Nuance Console (Business & Creator Portal):** The enterprise SaaS dashboard where researchers construct evaluation payloads, set budgets, and monitor SLA consensus.
3.  **Nuance Gateway (API & SDK Portal):** The developer interface where programmatic pipelines fetch aligned human preference weights.
4.  **Nuance Shield (Compliance & Moderation):** A dedicated administrative and automated safety moderation service.
