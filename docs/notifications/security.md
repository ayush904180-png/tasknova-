# Comms Security & RBAC Policies

Every tab panel and workflow action requires specific role clearance:

- **Owner & Super Admin**: Complete access.
- **Business**: Restricted to Campaigns, Templates, and Invoices.
- **Moderator**: Restricted to Announcements and Support threads.
- **Support**: Direct messaging and tickets.
- **Finance**: Billing logs and rule settings.
- **Developer**: Automation Rules, Templates, and GCP telemetry outputs.
- **Auditor**: Read-only GCP telemetry logs.
- **Read Only**: View-only mode.
