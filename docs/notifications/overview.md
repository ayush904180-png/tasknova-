# TaskNova Enterprise Notification & Engagement Platform

This module forms the unified communication node used by every microservice throughout the TaskNova AI Master Architecture.

## Features Overview

1. **Unified Alert Inbox**: Filterable categories, statuses, and priority flags.
2. **Support Messaging threads**: Pinned chats, read receipts, simulated typing loops, and image preview attachments.
3. **Broadcasting Hub**: Emergency banner alerts, product release notes, and dismissible pins.
4. **Logic-Based Automations**: Rule engine matching trigger events (IF) to specific actions (THEN) like mailing, pushing, or webhooks.
5. **Audience Campaigns**: Target filtering on Language, Role, and trust ratings with real-time conversion stats.
6. **Unified Ctrl+K Search**: Instant search matching across all notifications, templates, and active conversations.
7. **Offline Queue Sync**: Seamless offline-mode queuing that flushes pending state mutations to Google Firestore when reconnecting.
