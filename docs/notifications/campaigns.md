# Targeted Campaigns

Enables targeted, scheduled broadcasts to subset user bases.

## Targeting Segments

- Language (e.g. German, Spanish, Japanese, All)
- Target Role (e.g. Contributor, Business, Moderator)
- Minimum Trust rating slider
- Designated delivery channels (e.g. Email, SMS, Push, Slack)

## Operational Analytics

Upon publication, campaigns render performance stats:
- Audience size estimation
- Delivered ratios
- Average open rates
