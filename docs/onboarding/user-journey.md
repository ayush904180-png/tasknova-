# User Journey & Step Progression

The onboarding process comprises 8 sequential nodes engineered to gather user metrics seamlessly.

## Step Journey Definition

### Step 0: Welcome Screen
Introduces the user to the core platform values (Model Alignment, Secure Campaigns, Ledger Audits) with a highly readable visual display.

### Step 1: Choose Account Type
Prompts selection of one of three core account types (Contributor, Creator, Business). Admin options are completely isolated from this screen. Detailed estimated setup times are provided.

### Step 2: Accept Terms
Renders checkboxes for the four core legal agreements. A convenience "Accept All" button accelerates user progression.

### Step 3: Email Verification Status
Displays standard, reactive email state tracking. Includes developer-focused bypass buttons to trigger successful verification locally.

### Step 4: Basic Profile Setup
Captures the user's Display Name, Username, Country, Language, and Timezone. Includes a visual camera controller to cycle through beautiful avatar gradients.

### Step 5: Interests & Skills
Presents a search-enabled grid of selectable specialties (such as AI Evaluation, Translation, UI Testing).

### Step 6: Notification Preferences
Provides switches to adjust telemetry alert frequencies. High-contrast labels highlight safety critical options.

### Step 7: Completion Screen
Celebrates calibration completion and presents a dynamic summary card reflecting choices. Shows a primary CTA pointing to the home dashboard.
